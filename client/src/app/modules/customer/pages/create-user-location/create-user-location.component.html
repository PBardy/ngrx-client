Add Categories - Optional

Categories allow you to create a hierachical view of your locations, e.g. create you can create a category such as 'house' then add subcategories such as 'kitchen', 'pantry' and then add subcategories to these (e.g. 'top shelf')!

Add Tags - Optional

Tags are similar to categories, but allow for flat, arbitrary groups such as 'cold' or 'outside'. Tags are useful for general searching and can be combined to help you find exactly what your looking for, e.g. 'cold, inside and fancy' could be a wine cellar.
